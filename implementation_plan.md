# Implementation Plan - HearoPilot Landing Page

Build a premium, high-converting landing page for **HearoPilot**, an advanced AI-powered recording and transcription app.

## Goals
- Showcase the four core recording modes.
- Highlight AI-driven insights and task extraction.
- Create a stunning, modern visual experience (Dark mode, neon accents, glassmorphism).

## 1. Structure
- **Navigation**: Minimalist sticky header with "Features", "Modes", and "Download" button.
- **Hero Section**:
    - Animated text: "Your Meetings, Supercharged."
    - Visual of the app (placeholder/generated).
- **The 4 Recording Modes**:
    - Grid/Bento layout showing:
        - **Simple Listening**: Clean transcription.
        - **Short Meeting**: Focused highlights.
        - **Long Meeting**: Detailed summaries.
        - **Real-Time Translation**: Live multi-language support.
- **AI Insights & Tasks**:
    - Interactive "demo" section showing interleaved insights.
    - JSON task preview (simulated).
- **Footer**: Social links and newsletter.

## 2. Tech Stack
- **HTML5**: Semantic tags.
- **Vanilla CSS**: Modern CSS (Grid, Flex, Variables) for a premium look.
- **JavaScript**: Smooth scroll, scroll-triggered animations.
- **Google Fonts**: 'Inter' and 'Outfit' (premium typography).

## 3. Visual Aesthetic
- **Primary Color**: Deep Indigo (#0F0C29) to Dark Purple (#302B63).
- **Accent Color**: Electric Cyan (#00D2FF).
- **Components**: Glassmorphism (semi-transparent backgrounds with blur).
- **Animations**: Subtle fade-ins and parallax effects.
